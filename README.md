## About The Project

<img src="/images/screenshot.png" alt="" width="100%"/>

<h3>Some of the Features:</h3>
<ul>
  <li>Find information about Team Members through flashcards populated by a database.</li>
  <li>Toggle hide/show information about Team Members so users can quiz themselves.</li>
  <li>Search specific Team Members using a search bar that autocompletes names.</li>
  <li>Create sets of cards that are saved to the user's account.</li>
  <li>Dark mode, obviously.</li>
</ul>
